## Features

- Clean, responsive landing page
- Service description and features list
- Contact and sign-up form (or CTA)
- Pricing and plans (optional)
- Docs / FAQ section

## Getting started

1. Clone the repository:

   git clone https://github.com/Pshamaev/platon.law.git
2. Open the project in your preferred editor.
3. If the site is built with a static site generator (e.g., Hugo, Jekyll, Next.js), follow the project-specific setup in the respective folder.
4. Serve the site locally (example for a simple static site):

   npx http-server ./ -p 8080

   Then open http://localhost:8080 in your browser.
